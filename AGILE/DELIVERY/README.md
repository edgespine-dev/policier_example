# DELIVERY

This is the execution-trace workspace for agent-driven delivery.

## Where to look
- `BACKLOG/`: epic-level planning and scope.
- `STORIES/`: per-story specs, status, and slice progression.
- `PROTOCOL/`: communication contract between agents.
- `HANDOFFS/`: who handed off what, to whom, and why.
- `GATES/`: quality and integration gate outcomes.
- `ACTIVITY/`: chronological summary of agent actions.
- `EVIDENCE/`: artifacts that prove test/gate/decision outcomes.

## Why this structure
It keeps explainability and traceability in the build process itself:
- minimal context packets
- explicit handoff lineage
- explicit gate decisions
- durable evidence links
