# EVIDENCE

Store or reference files that justify decisions and gate outcomes.

Recommended layout:
- `EVIDENCE/<story-id>/tests/`
- `EVIDENCE/<story-id>/gates/`
- `EVIDENCE/<story-id>/notes.md`

Evidence should be linkable from:
- story files in `STORIES/`
- `HANDOFFS/HANDOFF_LOG.md`
- `GATES/GATE_LOG.md`
